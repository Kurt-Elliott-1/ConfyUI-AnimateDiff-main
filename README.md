# ComfyUI Restored — Portable Build for GTX 1060 6GB (Torch 2.1.2 cu118)

This repository preserves a fully working, portable ComfyUI environment
specifically configured for NVIDIA GTX 1060 6GB GPUs.  
It restores the last known stable setup compatible with Pascal cards,
using Python 3.10.0, Torch 2.1.2 cu118, and the classic early‑2024
ComfyUI node ecosystem.

## Why This Build Exists
Modern ComfyUI updates and managers dropped support for older GPUs and
introduced breaking changes. This portable build restores a known‑working
environment that runs reliably on GTX 1060 hardware without requiring
system‑wide installs or dependency conflicts.

## Features
- Portable — no installation required  
- Verified working on GTX 1060 6GB  
- Python 3.10.0 embedded  
- Torch 2.1.2 cu118 (last stable for Pascal GPUs)  
- Classic ComfyUI node set  
- Supports legacy workflows  
- Compatible with AnimateDiff classic (50MB plugin preserved separately)
